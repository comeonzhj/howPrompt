# howPrompt

一个驱动 Claude Code 拆解开源项目的提示词。

## 如何使用

### For Claude Code

1. 在`~/.claude`路径下创建`commands`文件
2. 下载 [howPrompt.md](howPrompt.md) 保存到`commands`文件中
3. 重启 Claude Code，使用`/howPrompt`指令即可直接激活使用

![claude code commands](/pic/claude.png)

## For Cursor

1. 在`~/.cursor`路径下创建`commands`文件
2. 下载 [howPrompt.md](howPrompt.md) 保存到`commands`文件中
3. 重启 Claude Code，使用`/howPrompt`指令即可直接激活使用
